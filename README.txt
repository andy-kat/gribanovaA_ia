Expense Tracker
Автор: Грибанова Анжелика Евгеньевна

Описание
Expense Tracker — это простое приложение для учёта личных расходов, написанное на Python с использованием библиотеки tkinter. Программа позволяет добавлять расходы, фильтровать их по категориям и датам, а также видеть итоговую сумму за выбранный период. Все данные сохраняются в файл data/expenses.json.

Возможности
Добавление расходов с указанием суммы, категории и даты.
Встроенные категории: Продукты, ЖКХ, Транспорт, Здоровье, Развлечения.
Фильтрация расходов по категории и диапазону дат.
Автоматический подсчёт общей суммы расходов за выбранный период.
Сохранение и загрузка данных из JSON-файла.
Требования
Python 3.6+
Библиотека tkinter (входит в стандартную поставку Python).
Установка и запуск
Убедитесь, что у вас установлен Python 3.
Скачайте или клонируйте этот репозиторий.
Запустите скрипт:
bash
Копировать
python expense_tracker.py
Использование
Добавить расход:
Введите сумму (только положительное число).
Выберите категорию из списка.
Укажите дату в формате ГГГГ-ММ-ДД (например, 2026-04-30).
Нажмите «Добавить расход».
Фильтрация:
Используйте выпадающий список для выбора категории.
Укажите начальную и конечную даты.
Нажмите «Обновить» для применения фильтра.
Просмотр суммы:
После фильтрации или при запуске программы появится окно с итоговой суммой расходов за выбранный период.
Структура проекта
Копировать
expense_tracker/
│
├── expense_tracker.py      # Основной файл приложения
├── data/
│   └── expenses.json       # Файл для хранения данных о расходах
└── README.md               # Этот файл
Примечания
Если файл data/expenses.json отсутствует, он будет создан автоматически при первом запуске.
В случае ошибок в формате данных (например, некорректная дата или сумма), программа покажет сообщение об ошибке.
